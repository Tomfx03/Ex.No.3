# Aim

To design prompts using different prompt engineering techniques and evaluate how each prompting method generates responses for a specific scenario.

# Explanation

Prompt engineering is the process of designing effective prompts to guide AI models to generate accurate and meaningful outputs. Different prompt styles can influence how the AI understands and answers a question.

In this experiment, four types of prompting techniques are used:

• Straightforward Prompt
• Tabular Format Prompt
• Preceding Question Prompt
• Missing Word Prompt

These prompts are applied to a use case from the Education domain to observe how AI generates responses based on different prompt structures.

# Procedure
# 1. Straightforward Prompts
# Definition

A straightforward prompt is a simple and direct instruction given to the AI model without additional structure or conditions.

# Use Case Example

Domain: Education Technology

# Prompt

"What are the benefits of online learning?"

# Output

Online learning offers flexibility, allows students to learn at their own pace, provides access to global resources, and reduces travel time and costs.

# Prompt

"List four advantages of using digital classrooms."

# Output

• Easy access to study materials
• Interactive learning using multimedia
• Better communication between teachers and students
• Ability to attend classes from any location

# 3. Preceding Question Prompting
# Definition

Preceding Question Prompting uses a sequence of connected questions. Each question helps build a deeper understanding of the topic.

# Use Case Example

Domain: Information Technology

# Prompt

"What is Artificial Intelligence?"

# Output

Artificial Intelligence (AI) is a technology that enables machines to simulate human intelligence and perform tasks such as learning, reasoning, and problem solving.

# Prompt

"Why is Artificial Intelligence important in modern technology?"

# Output

Artificial Intelligence is important because it improves automation, increases efficiency, supports data analysis, and helps in developing intelligent systems such as chatbots, recommendation systems, and self-driving cars.

4. Missing Word Prompting
# Definition

Missing word prompting provides an incomplete sentence and asks the AI to fill in the missing word.

# Prompt

"The largest planet in our solar system is ______."

# Output

Jupiter

Prompt

"The programming language widely used for web development is ______."

# Output

JavaScript

# Conclusion

In this experiment, different prompt engineering techniques were applied to the education and technology domains. Each prompting method produced different styles of responses.

Straightforward prompts generated quick answers, tabular prompts produced structured data, preceding question prompts provided detailed explanations, and missing word prompts produced short and precise responses.

This experiment demonstrates how prompt design plays an important role in improving AI-generated outputs
